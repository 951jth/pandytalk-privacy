# 계정 삭제 및 데이터 삭제 안내

이 페이지는 **[앱 이름]** 이용자가 계정 및 관련 데이터를 삭제하는 방법을 안내하기 위해 제공됩니다.  
현재 **앱 내 프로필 화면에서 회원 탈퇴 기능을 통해 직접 삭제 요청이 가능하며**, 필요한 경우 이메일을 통해서도 요청하실 수 있습니다.

---

## 1. 앱 내에서 계정 삭제(회원 탈퇴) 방법

1. **[앱 이름] 앱 실행**
2. 하단 메뉴 또는 상단 메뉴에서 **`프로필` 탭** 또는 **`설정` 화면**으로 이동
3. **`회원 탈퇴` 버튼 선택**
4. 안내 문구 확인 후 **`확인` 또는 `탈퇴하기` 버튼** 클릭

> ※ 앱 업데이트에 따라 버튼 위치나 명칭이 달라질 수 있습니다.

---

## 2. 계정 삭제 시 처리되는 데이터

회원 탈퇴 완료 후 아래 정보는 삭제되거나 복구할 수 없는 상태로 처리됩니다.

- 회원 기본 정보 (이메일, 닉네임 등)
- 개인 프로필 정보 (사진, 상태 메시지 등)
- 푸시 알림용 기기 토큰 (예: Firebase FCM)

### 📌 추가적으로 처리될 수 있는 항목

- 채팅 또는 게시글 등 **다른 이용자와의 상호작용 기록은 삭제되지 않으며**,  
  작성자 ID가 **익명화 처리**되거나 관련 법령에 따라 최소 기간 보존될 수 있습니다.

---

## 3. 관련 법령에 따른 보존 데이터

관련 법령에 따라 아래 정보는 일정 기간 보존될 수 있습니다.

| 보존 항목 | 보존 기간 | 근거 법령 |
|-----------|-----------|-----------|
| 거래/결제 기록 | 5년 | 전자상거래 등에서의 소비자보호에 관한 법률 |
| 소비자 불만 또는 분쟁 처리 기록 | 3년 | 전자상거래 등에서의 소비자보호에 관한 법률 |
| 접속 이력(IP 등) | 3개월 | 통신비밀보호법 |

> 보존 기간 경과 후에는 안전한 방법으로 완전히 삭제됩니다.

---

## 4. 앱 내에서 탈퇴가 어려운 경우 (이메일 요청 가능)

앱 내에서 탈퇴가 불가능하거나 추가 데이터 삭제 요청이 필요한 경우 아래 이메일로 요청해 주세요.

**📧 이메일: `support@your-domain.com`**

메일 제목 예시:

```
[계정 삭제 요청] 홍길동 / user@example.com
```

메일 내용 예시:

```
앱 이름: [앱 이름]
사용자 이메일 또는 UID: user@example.com
요청 내용: 계정 및 관련 데이터 삭제 요청
추가 문의사항: (선택)
```

---

## 5. 처리 기간 및 유의사항

- 삭제 요청은 **영업일 기준 7일 이내** 처리됩니다.
- 삭제 또는 익명 처리된 데이터는 **복구가 불가능**합니다.
- 동일한 이메일 또는 계정 정보로 **재가입 시 기존 데이터는 복원되지 않습니다.**

---

## 6. 문의

계정 삭제와 데이터 처리 관련 문의는 아래 연락처로 문의해 주세요.

- 이메일: **support@your-domain.com**
- 운영 시간: 평일 10:00 ~ 18:00 (점심 12:00 ~ 13:00)

---

## 🔐 참고 사항

- 계정 삭제 전에 필요한 데이터(채팅 내용, 이미지 등)는 **직접 백업 후 진행해 주세요.**
- 법적 보관 기준이 존재하는 정보는 **정해진 보존 기간 후 자동 삭제됩니다.**
